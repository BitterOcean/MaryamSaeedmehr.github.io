Description

I worked on this unique project that combines education and gamification as a freelance Full Stack Web Developer. This web application handles students' homework assignments and features a real-time fighting game to encourage students to solve more homework and earn virtual coins. My expertise lies in utilizing Django Rest Framework (DRF) with Channels and Celery for the backend, along with React.js and Material-UI (MUI) for the front-end development.
